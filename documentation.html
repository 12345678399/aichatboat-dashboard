Introduction

Welcome to NebulaX, a premium, high-performance admin dashboard template built with vanilla HTML, CSS, and JavaScript. It is designed to be lightweight, fast, and easy to customize without the overhead of heavy frameworks.

Key Features

  • No Framework Dependency: Pure Vanilla JS and CSS.
  • Modern Design: Glassmorphism, dark mode ready, and premium aesthetics.
  • Fully Responsive: Mobile-first approach.
  • SPA-like Experience: Fast navigation without page reloads.
  • Comprehensive Layouts: Vertical sidebar and Horizontal top navigation.
  • Themeable: Easy color customization via CSS variables.

Installation

Getting started with NebulaX is incredibly simple since it doesn't require a build process (unless you want to add one).

Steps

  1. Unzip the downloaded package.
  2. Open the folder in your favorite code editor (VS Code recommended).
  3. Open index.html in your browser.

That's it! You are ready to go.

Note: For the best experience and to avoid CORS issues with some local file requests (if you add them manually), it is recommended to run the project on a local server (e.g., using Live Server extension in VS Code).

Theming & Colors

NebulaX uses CSS Custom Properties (variables) for easy theming. You can find all color definitions in css/variables.css.

Changing the Primary Color

To change the primary brand color, simply edit the --primary variable:

:root {
    /* Change this value */
    --primary: #2563EB; 
    
    /* Optional: Update shades */
    --primary-hover: #1D4ED8;
}

Dark Mode

Dark mode is implemented using a [data-theme="dark"] attribute on the <html> element. The colors for dark mode are defined in the same variables.css file under the dark theme block.

Layout Options

NebulaX supports two main layouts: Vertical Sidebar (default) and Horizontal Menu.

Switching Layouts

Users can switch layouts using the layout toggle button in the header. This preference is saved in localStorage.

To force a default layout, you can modify the initLayoutSwitcher function in js/main.js.
